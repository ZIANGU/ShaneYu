# ShaneYu

一款为开发者打造的现代博客主题，关于专注与极致的分享空间。

## 功能特性

- 🌙 **深色/浅色主题切换** - 支持自动切换和手动切换
- 🎨 **主题背景个性化** - 支持图片/视频壁纸作为主题背景
- 🪟 **毛玻璃效果** - 壁纸模式下全站元素添加半透明磨砂效果
- 📱 **响应式设计** - 完美适配大屏幕、中等屏幕和移动端
- 📊 **文章数据统计** - 标签页面支持数据统计图表
- 🔍 **搜索功能** - 支持快捷键搜索
- 📤 **分享功能** - 支持微信、微博、QQ空间分享

## 技术栈

- Halo 2.0+
- Thymeleaf
- TypeScript
- Vite
- Tailwind CSS (CSS Variables)

## 安装

1. 下载主题包 `halo-theme-ShaneYu.zip`
2. 在 Halo 后台管理界面 -> 外观 -> 主题 -> 安装主题
3. 上传主题包并启用

## 配置

主题设置包含以下选项：

- 基本设置：站点标题、副标题、Logo
- 导航设置：主菜单配置
- 背景设置：图片/视频壁纸
